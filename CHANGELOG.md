# Changelog

## Version format `vM.m.T`

| Segment | Meaning |
|--------|---------|
| **M** | Development milestone (product phase; bump when the milestone changes). |
| **m** | Build iterations within the current milestone (incremented automatically after each successful full workspace `npm run build`). |
| **T** | Unit test iteration counter (increment with `npm run version:test-iteration` when you complete a deliberate test pass / suite change). |

The value in root `package.json` `version` must stay in sync with this log for dashboards.
